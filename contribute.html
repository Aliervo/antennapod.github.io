{% t titles.contribute %}

AntennaPod is under active development by volunteers. You can contribute too, with code or with comment!

  • You speak any other language than English? Help us translate AntennaPod via Transifex
  • You can explain well? Help other users on our forum
  • You are good at breaking stuff? Become a beta tester
  • You can program in Java? Help test pull requests or submit new features or bug fixes on GitHub. You can also visit our developer wiki for more information about how to build AntennaPod.
